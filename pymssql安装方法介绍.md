# pymssql安装方法

## 一、打包好的安装方法：

1. pymssql包：pymssql-2.1.3-cp36-cp36m-win_amd64.whl  
  
2. 把它放到python安装路径的script文件夹下    
3. 在cmd中输入 pip install pymssql-2.1.3-cp36-cp36m-win_amd64.whl  

文件下载路径：tfs\TestManager\tools\installation_packages\pymssql安装文件  

## 二、通过网站下载安装：

1. 下载pymssql包：pymssql-2.1.3-cp36-cp36m-win_amd64.whl  
http://www.lfd.uci.edu/~gohlke/pythonlibs/  

2. 把它放到python安装路径的script文件夹下  

3. 在cmd中输入 pip install pymssql-2.1.3-cp36-cp36m-win_amd64.whl
